Satellite Visualizer Web App

This is a simple prototype of a web app that helps people see satellites flying above them in real-time using a 3D globe. I’m using the N2YO Satellite API to fetch live satellite positions based on the user’s location. The goal is to make space more accessible and visible, especially to people who may have never looked up at the stars with purpose before.

What It Will Do

  • Show a 360° sky based on the user’s location
  • Plot visible satellites as floating points with labels
  • Update in real-time using live API data

How I’ll Make It

  • Frontend: HTML, CSS, JavaScript, Three.js
  • API: N2YO Satellite Tracking API
  • Data: User geolocation, satellite positions, names
